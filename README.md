Advanced Java Assignments (4, 5, 6)
Overview

This project demonstrates important Object-Oriented Programming (OOP) concepts in Java including:

Abstract Classes

Inheritance and Method Overriding

Interfaces and Polymorphism

Assignment 4 – Shape Hierarchy

Package: assignment4_shapes

An abstract class Shape defines area() and perimeter() methods.
Classes Circle, Rectangle, and Triangle extend Shape and provide their own implementations.

Concepts used:

Abstraction

Method Implementation

Polymorphism

Assignment 5 – Vehicle Start Behavior

Package: assignment5_vehicles

Class Vehicle contains a start() method.
Classes Car and Bike extend Vehicle and override the start() method.

Concepts used:

Inheritance

Method Overriding

Runtime Polymorphism

Assignment 6 – Employee Compensation

Package: assignment6_employees

Interface Payable defines calculatePay() method.
Classes SalariedEmployee and ContractEmployee implement this interface.

Concepts used:

Interface

Abstraction

Polymorphism

How to Run

Run Main.java to execute all three assignments.
